[h1]Compatibility Patch for Invictus Mod[/h1]

In your Imperator launcher, when enabling mods, the order should be:
[olist]
 [*] Legion Distinctions Overhaul
 [*] Invictus
 [*] Legion Distinctions Overhaul: Invictus Patch
[/olist]
